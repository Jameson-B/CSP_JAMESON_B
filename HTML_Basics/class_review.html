Computer Science Principles is an introductory course to the field of computer science, covering topics from hardware to programming to the Internet.

Basics

What is a computer?
A computer is anything that stores and manipulates information.
Hardware and Software
Hardware is the physical components of the computer.
Software is the instructions that the computer runs on.
Binary
A base-2 number system.
Made of 0 and 1.
One digit represents a power of 2.
Ex: 0101 = 5
Data Compression
Lossy compression is when some of the data is lost in compression.
Suitable for images, audio.
Lossless compression is when none of the data is lost, and can be completely reconstructed.
Suitable for text documents, programs.

Variables
A variable is a place where data is stored.
A variable declaration is the assigning of a value to a variable, using (in Python) the assignment operator =.
Some common data types include: integer, float, string, and Boolean.
An integer is a whole number, including negatives, positives, and zero.
A float is a number containing decimals.
A string is a sequence of characters.
A Boolean, first conceptualized by English mathemtician George Boole, has the value of true or false.
Expressions
An expression is a statement of operators and operands.
An operator is what performs an action on the data.
An operand is the data that is being acted on.
The basic arithmetic operators are (in Python): addition +, subtraction -, multiplication *, exponentiation **, floor division //, and modulus %.
Strings
A string is a sequence of characters.
Similar to arithmetic operators for numbers, there are string operators for letters and words.
Functions
A function (also known as a procedure or a method) is a chunk of code that takes parameters, operates on them, and returns an output.
A function definition is the creation of a function. It consists of a keyword, such as def (in Python), the function name, the parameters, the content of the function, and a return statement (in some cases).
A function invocation (also known as "calling a function") is the usage of a function in a program. it consists of the function name and the number of required arguments.
A parameter is a value that is operated on in the function. A parameter is like a local variable within the function.
An argument is the data that is given to a parameter when a function is called within a program.
A return statement assigns the output of the function to the function invocation. Essentially, it tells the program what the answer to the function is.
