Computer Science Principles is an introductory course to the field of computer science, covering topics from hardware to programming to the Internet and beyond.

Basics

What is a computer?
A computer is anything that stores and manipulates information.
Hardware and Software
Hardware is the physical components of the computer.
Software is the instructions that the computer runs on.
Binary
A base-2 number system.
Made of 0 and 1.
One digit represents a power of 2.
Ex: 0101 = 5
Data Compression
Lossy compression is when some of the data is lost in compression.
Suitable for images, audio.
Lossless compression is when none of the data is lost, and can be completely reconstructed.
Suitable for text documents, programs.

Programming

Languages Taught:
Python
C
Variables
A variable is a place where data is stored.
A variable declaration is the assigning of a value to a variable, using (in Python) the assignment operator =.
Some common data types include: integer, float, string, and Boolean.
An integer is a whole number, including negatives, positives, and zero.
A float is a number containing decimals.
A string is a sequence of characters.
The Boolean, attributed to English mathemtician George Boole, has the value of true or false.
An array (called a list in Python) is a sequence of variables, stored in one place.
Expressions
An expression is a statement of operators and operands.
An operator is what performs an action on the data.
An operand is the data that is being acted on.
The basic arithmetic operators are (in Python): addition +, subtraction -, multiplication *, exponentiation **, floor division //, and modulus %.
Strings
A string is a sequence of characters.
Similar to arithmetic operators for numbers, there are string operators for strings.
Functions
A function (also known as a procedure or a method) is a chunk of code that takes parameters, operates on them, and returns an output.
A function definition is the creation of a function. It consists of a keyword, such as def (in Python), the function name, the parameters, the content of the function, and a return statement (in some cases).
A function invocation (also known as "calling a function") is the usage of a function in a program. it consists of the function name and the number of required arguments.
A parameter is a value that is operated on in the function. A parameter is like a local variable within the function.
An argument is the data that is given to a parameter when a function is called within a program.
A return statement assigns the output of the function to the function invocation. Essentially, it tells the program what the answer to the function is.
Conditionals
A conditional is a block of code that only runs when a certain condition is met.
It is also known as an if-then statement.
A conditional works by first checking if a condition is met. If it evaluates to either true, meaning that the condition has been met, or false, indicating that it has not.
Loops
A loop is a block of code that repeats a certain number of times.
There are two types of loops: while loops and for loops.
A while loop repeats as long as a certain condition evaluates to true. While loops usually invole an iterator, a variable like x or i that counts the number of iterations in a loop by being the subject of a condtional statement, such as x == 10.
An iteration is defined as one cycle of a loop.
In a while loop, the iterator changes on each iteration. Eventually, the conditional at the beginning of the while loop will no longer be true and the loop will terminate.
A for loop repeats the number of times it is told to by the program.
An array can be used to instruct a for loop.
For example, if you have defined a list called fruits, using it in a for loop would look like (in Python): for fruit in fruits: (do something). The word fruit in our code just defines one item in the list fruits. We could call this item anything, from fish to abc, but since this is a list of fruits, it makes sense to call one item in this list a fruit.
To exit a loop prematurely, the keyword (in Python) break is used. This stops the loop wherever it is and moves onto the code outside the loop.
If you want to move onto the next iteration of a loop, use the keyword (in Python) continue.
SQL
Pronounced "sequel", SQL stands for "Structured Query Language".
SQL is designed for retrieving and organizing large amounts of data.
SQL is applied to managing data for governments, corporations, and the like.

The Internet

Networking
A network is a system of interconnected computers that share data and send data.
The Internet
The Internet is a global network of networks.
Information is sent across the Internet in small pieces called packets.
The World Wide Web
The World Wide Web (WWW) is a system of connected documents that run on servers distributed all over the planet.
HTML Basics
HTML stands for "Hypertext Markup Language". Rather than a programming language, which writes commands, HTML is for structuring web pages.
Every element in HTML is contained within a tag, surrounded by angle brackets <>.
This web page was written with HTML.
HTML Advanced
The unit this assignment is for.
HTML5, the fifth edition of HTML, includes 145 different tags.
HTML is used in combination with another language called CSS (Cascading Style Sheets), which styles the elements on the page, changing, size, color, shape, and so forth. CSS is what makes web pages look pretty.
